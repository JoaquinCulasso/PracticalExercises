Maarcaadores de Fuentes

B: Se utiliza para poner en negrita al texto.

BIG: Se utiliza para convertir a una fuente más grande esto es interpretado por el navegador.

I: Se utiliza para convertir el texto a cursiva.

S: Se utiliza para agregar una raya al texto.

SMALL: Se utiliza para hacer el texto más pequeño.

STRIKE: Es el mismo resultado que s, pone un raya en el texto.

TT: Se utiliza para dar formato de teletipo como se utiliza en los monitores o impresoras antiguas.

U: Se utiliza para subrayar una palabra.
